Class MamProvider<T>

mamprovider<t>

is the main class encapsulating the messaging with the tangle. It greatly simplifies interaction with the DLT, while also caching data locally, so that not all calls are translated into actual API invocations.

Generic is for the data type to be published (e.g. an Interface). You can set it to when working with multiple data types.
